# オンライン個別指導塾 学ジム

英語・国語・総合型選抜の個別指導を提供するオンライン塾のウェブサイトです。

## 📋 概要

学ジムは、以下のサービスを提供しています：

- 📚 大学受験対策（志望校特化）
- 🏆 英検・漢検・数検等の検定対策
- 🎓 公募制推薦・総合型選抜対策
- 👨‍🏫 個別オンライン指導
- 🤝 業界影響力を持つ厳選講師の紹介

## 🛠️ 技術スタック

- **HTML5** - セマンティックマークアップ
- **CSS3** - レスポンシブデザイン
- **JavaScript (Vanilla)** - 動的機能実装
- **アクセシビリティ対応** - WCAG準拠

## 🚀 使い方

### ローカルで実行

```bash
# リポジトリをクローン
git clone https://github.com/masuroom0601-lab/gakujim.git
cd gakujim

# ブラウザで開く（例：VS CodeのLive Serverを使用）
# または、ローカルサーバーで実行
python -m http.server 8000
# ブラウザで http://localhost:8000 にアクセス
```

## 📋 フォーム検証

### バリデーション項目

- **お名前**: 必須、空白チェック
- **ふりがな**: 必須、空白チェック
- **メールアドレス**: 必須、形式チェック
- **電話番号**: 必須、形式チェック（10文字以上）
- **お問い合わせ内容**: 必須、空白チェック
- **同意チェック**: 必須
- **CAPTCHA**: 必須、正答チェック（答え: 45H2）

### エラーメッセージ

フォーカスが失われたときと送信時にリアルタイム検証が実行されます。

## 📱 レスポンシブ対応

- **PC** (1200px以上): フルレイアウト
- **タブレット** (768px～1199px): 調整済みレイアウト
- **スマートフォン** (480px～767px): 最適化レイアウト
- **小型デバイス** (479px以下): コンパクトレイアウト
